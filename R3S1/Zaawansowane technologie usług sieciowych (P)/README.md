## Project: MagazynierUZ

A warehouse management system — a web application for managing warehouses, storage locations, products, and real-time inventory monitoring.

### Repositories

- **[MagazynierUZ-backend](https://github.com/Guliveer/MagazynierUZ-backend)** — Backend built with **Java (Spring Boot)**. Provides a REST API with JWT authentication, role-based access control (RBAC), multi-tenant organization support, low stock alerts, and PDF report exports.

  ![Java](https://img.shields.io/badge/Java-Spring_Boot-green?logo=springboot)

- **[MagazynierUZ-frontend](https://github.com/Guliveer/MagazynierUZ-frontend)** — Frontend built with **TypeScript** using **Next.js 16** and **React 19**. Features a responsive UI with multi-language support (PL/EN), light/dark theme, interactive warehouse maps, statistics charts, and an admin panel.
