# Podstawy programowania na platformę Android (L)

## Project: Ecological Assistant

- **[Ecological-Assistant](https://github.com/Guliveer/Ecological-Assistant)** — An Android app written in **Java** for eco-friendly product and packaging management. Features product and packaging tracking, a deposit/return system, report generation (monthly, by category, expired products), data export to CSV/PDF/HTML, and a return points map. Built with Room database and MVVM architecture.
