## Project: RFID Access Control System

- **[RFID-access-manager](https://github.com/Guliveer/RFID-access-manager)** — A complete RFID-based access control system combining embedded hardware (**ESP32-C3** + RFID reader) with a cloud-managed **Next.js 16** web dashboard. Enables secure management of physical access to doors, gates, and restricted areas with real-time HTTPS verification, role-based access control, full audit logging, and a responsive UI with dark mode. Backend powered by **Supabase (PostgreSQL)**.
